สื่อการสอนหัวข้อ Python

แบบทดสอบก่อนเรียน

ภาษา Python เป็นภาษาที่ได้รับอิทธิพลมาจากภาษา C โดยเวลาที่แปลงไปเป็นภาษาเครื่อง จำเป็นต้องแปลงกลับไปเป็น C แล้วค่อยแปลง C นั้นไปเป็นภาษเครื่องอีกทีหนึ่ง นั่นเป็น เหตุหนึ่งที่ทำให้ภาษา Python เป็นภาษาที่ทำงานได้ช้ากว่าภาษาอื่นๆ มาก เพราะว่ามีขั้นที่ต้องแปลงมากกว่าภาษาอื่น แต่สิ่งที่ได้มาจากการทำแบบนี้ คือ ภาษา Python มีความ ใกล้เคียงกับภาษาที่มนุษย์ใช้งานมากที่สุด ทำให้เป็นภาษาที่เรียนรู้ได้ง่าย และเร็วที่สุด ภาษา Python ถูกนำไปใช้งานอย่างกว้างขว้างในอุตสหากรรมต่างๆ ทั้งในการพัฒนาเว็บเซิฟเวอร์ การพัฒนาเกม และอื่นๆ ภาษา Python สามารถเขียนได้หลากหลาย ปัจจุบันเวอร์ของภาษา Python คือ Python 3.10.x

ส่วนอธิบาย

  • print('''Text''') คือ ฟังก์ชั่นสำหรับการแสดงผลข้อความออกทาง Terminal แบบที่ปรับได้มากกว่า
  • def variableName(): คือ การประกาศฟังก์ชั่นในภาษา Python
  • print(f'''Text{variable}''') คือ การแสดงผลงข้อความผ่าน Terminal แบบ Format String
  • for in คือ คำสั่งวนซ้ำ ทำให้สามารถทำงานซ้ำๆ ได้หลายครั้งตามเงื่อนไขที่กำหนด
  • variableName = buildInFunc คือ การประกาศตัวแปร เพื่อเก็บค่าลงไปในตัวแปรนั้น
  • functionName() คือ การสั่งรันด์ฟังก์ชั่นที่สร้างขึ้น
  • int() คือ การรับค่าเป็ฯจำนวนเต็มเท่านั้น
  • input() คือ การรับค่าเข้าจากผู้ใช้งานโปรแกรม
  • -> โค้ดด้านบนเป็นการใช้งานลูป เพื่อหาค่าของเลขที่คุณกัน แล้วแสดงผลออกมา โดยการจะแสดงผลจะเริ่มนับจากเลข 0 เสมอ
  • -> ด้ายซ้าย คือ ส่วนที่ใช้ในการเขียนโค้ด ด้านขวา คือ Terminal ที่ใช้ในการแสดงผลการรันด์ของโค้ด
  • -> Online Code Editor ตัวนี้ชื่อ replit
