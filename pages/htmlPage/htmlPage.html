สื่อการสอนหัวข้อ HTML

แบบทดสอบก่อนเรียน

ภาษา HTML เป็นภาษาที่ใช้สำหรับการรร่างโครงร่างอย่างที่ได้กล่าวไว้ ที่ด้านนี้ จะเรียกว่า การ์ด หรือในภาษาอังกฤษว่า Card การ์ดจะเอาไว้ใช้สำหรับใส่ข้อความเพื่อนนำเสนอสินค้า และบริการบนหน้าเว็บไซต์ ในโครงสร้างของภาษา HTML จะมีใช้ Tag เปิดและ Tag ปิดในการกำหนดขอบเขตต่างๆ ของเนื้อ โดยที่ Tag เปิดจะไม่มีเครื่องหมาย / แต่ Tag ปิดมี และชื่อเต็มๆ ของภาษา HTML คือ Hypertext Markup Language

ส่วนอธิบาย

  • div tag : จะใช้สำหรับทำกล่องให้กับ Element ต่างในหน้าเว็บไซต์
  • img tag : จะใช้สำหรับการรูปภาพให้กับหน้าเว็บไซต์
  • h2 tag : จะใช้สำหรับกำหนดหัวเรื่องต่างๆ โดยจะมีทั้งหมด 6 ตัว ได้แก่ h1-h6 โดย h1 จะมีขนาดที่ใหญ่ที่สุด และ h6 จะมีขนาดที่เล็กที่สุด
  • p tag : จะใช้วำหรับการกรอบตัวอักษร แต่ในทางปฏิบัติแล้ว เราสามารถพิมพ์ข้อโดยที่ไม่มี Tag ใดๆ ครอบอยู่เลยก็ได้
  • -> ในภาษา HTML นั่น ยังมี Tag อีกมากมาย ซึ่งผู้จัดทำได้ทำการแนบลิงค์สำหรับการศึกษาเพิ่ม แหล่งข้อมูลนั่นจะเป็นประโยชน์แก่/แด่คนทุกคนที่ต้องการเป็นนักพัฒนา
