สื่อการสอนหัวข้อ CSS

แบบทดสอบก่อนเรียน

ภาษา CSS มีชื่อเต็มว่า "Cascading Style Sheets" เป็นภาษาที่ใช้งานคู่กับภาษา HTML โดยที่ภาษา CSS จะใช้สำหรับตกแต่งหน้าเว็บไซต์ให้มีความสวยมากขึ้น ภาษา CSS เป็นภาษาทีแยกออกมาจากภาษา HTML ไม่ใช่ภาษาเดียวกัน และเป็นภาษาที่เป็นกึ่งโปรแกรมมิ่งอยู่บ้าง โดยสามารถกำหนดเงื่อนไขอย่างง่าย เพิ่มตัวแปร และ การคำนวณอย่างง่ายได้ จึงเป็นภาษาที่มีเป็นโปรแกรมมิ่งอยู่บ้าง แต่ไม่ได้เหมือนภาษา C และภาษา Python ที่เป็นภาษาโปรแกรมมิ่งเต็มรูปแบบ

ส่วนอธิบาย

  • display : เป็นส่วนที่กำหนดรูปแบบการแสดงผลบนหน้าเว็บไซต์ หรือบน Web Browser
  • background : เป็นส่วนที่กำหนดสีพื้นของ Element ต่างในหน้าเว็บไซต์(ส่วนนี้ใช้งานเพื่อทดสอบความกว้าง และการมีอยู่ของ Element เท่านั้น)
  • box-shadow : เป็นส่วนที่สร้างเงาให้กับ Element
  • justify-content : เป็นตัวที่ใช้กำหนดการจัดรูปแบบที่ใช้งานใน display ประเภท flexbox
  • position : เป็นส่วนที่ใช้สำหรับกำหนดรูปแบบของตำแหน่งของ Element ในหน้าเว็บไซต์
  • margin : เป็นส่วนที่ใช้สำหรับกำหนดความห่างกันระหว่าง Element(ส่วนนี้จะเข้าใจได้ต้องศึกษาในส่วนของ CSS Box Model)
  • padding : เป็นส่วนที่ใช้สำหรับการกำหนดความห่างของเนื้อหาที่อยู่ใน Element(ส่วนนี้จะเข้าใจได้ต้องศึกษาในส่วนของ CSS Box Model)
  • border-radius : เป็นส่วนที่ใช้สำหรับทำให้ขอบมุมต่างๆ โค้งมนมากขึ้น
  • text-align : เป็นส่วนที่ใช้สำหรับกำหนดให้ข้อความ รุปภาพ หรือ Element ที่เกี่ยวข้องกับ Font อยู่ในตำแหน่งที่ต้องการ เช่น ตรงกลาง
  • width : เป็นส่วนที่ใช้สำหรับกำหนดความกว้างให้กับ Element ใช้ได้กับทุกๆ Element ที่ต้องการปรับความกว้าง และยังนิยมใช้งานกับรูปภาพอีกด้วย
  • font-size : เป็นส่วนที่ใช้สำหรับกำหนดขนาดของ Font หรือตัวอักษร
  • -> จากส่วนที่แล้ว(HTML) ตัวเครื่องหมายที่ใช้กำหนด class คือ เครื่องหมาย "." ส่วนตัวที่ไม่ได้มีจุดอยู่ด้านหน้า หมายความว่าเลือกที่ Element โดยตรง
