Hypertext Markup Language

     ภาษา HTML หรือ Hypertext Markup Language เป็นภาษาที่ใช้สำหรับการวางโครงร่างของหน้าเว็บไซต์ ลักษณะของหน้าเว็บไซต์ที่เขียนโดย HTML ก็จะเป็นลิงค์สีฟ้าๆ และมีสีตัวอัการนิดหน่อย มีตารางอย่างง่าย ให้นึกถึงเว็บไซต์ที่ ถูกพัฒนาในยุค IE/Internet Explorer

Cascading Style Sheets

     ภาษา CSS เป็นภาษาที่ใช้คู่กับ HTML ใช้สำหรับการตกแต่ง หรือสร้างความสวยงามให้กับ HTML นอกจากจะใช้งานเพื่อความสวยงาม ยังในการจัดรูปแบบอีกด้วย ในภาษา CSS จะมีรูปแบบการจัดรูปแบบหลากหลาย และทำให้หน้าเว็บไซต์เรียบร้อยขึ้น ถ้าเปรียบ HTML เป็นทั้งหลัง CSS ก็สี และของตกแต่งที่ให้บ้านน่าอยู่

C Language

     ภาษา C เป็นภาษาที่มีความเก่าแก่มากที่สุด ภาษา C เหมาะกับการเขียนโปรแกรมที่สามารถทำงานได้ในหลากหลายอุปกรณ์ เพราะว่าเป็นภาษาที่มีขนาดเล็ก ใช้ Memory น้อยกว่าภาษาอื่นๆ และภาษา C ยังเป็นพื้นฐานให้กับหลายๆ ภาษาในปัจจุบันอีกด้วยเรียกได้ว่าเป็นภาษาที่มีความสำคัญอย่างมาก และปัจจุบันก็ยังมีการใช้งานในหลากหลายอุตสาหกรรมแม้ว่าจะผ่านมานานมากแล้วก็ตาม

Python

ภาษา Python เป็นภาษาที่มีพื้นฐานมาจากภาษา C หรือได้รับอิทธิพลมา ภาษา Python สามารถทำงานได้หลากหลาย และเป็นภาษาที่เรียกได้ว่าเขียนได้ง่ายที่สุดแล้วในบรรดาภาษาทางคอมพิวเตอร์ทั้งหมดบนโลก ภาษา Python ใช้งานได้หลากหลายมาก เช่นการพัฒนาเกม การพัฒนาเว็บเซิฟเวอร์ และการเขียนโปรแกรมอื่นๆ แต่ Python มีข้อเสียหลักอยู่หนึ่งอย่าง คือ ความช้าในการทำงาน และประมวลผล
